# 网络相关

## 服务器连接

**为了减少公网带宽占用，有校内地址的情况下，推荐优先选择校内地址。**

**除此之外，校内隧道可获得更稳定，低延迟的连接。**

**校外建议选择就近的地址使用，若某地址无法连接可换用其他地址。**

| 地址                    | 地域                     | 当前状态 |
| :---------------------- | :----------------------- | :------- |
| `local.permod.hitmc.cc` | 校内-本部                | ![status](https://monitor.fp.zsh2517.com/badge/local.permod.hitmc.cc/_updown)   |
| `n.permod.hitmc.cc`     | 公网-山东枣庄（樱花frp） | ![status](https://monitor.fp.zsh2517.com/badge/n.permod.hitmc.cc/_updown)   |
| `s.permod.hitmc.cc`     | 公网-浙江宁波（樱花frp） | ![status](https://monitor.fp.zsh2517.com/badge/s.permod.hitmc.cc/_updown)   |
| `gz.permod.hitmc.cc`    | 公网-广州                | ![status](https://monitor.fp.zsh2517.com/badge/gz.permod.hitmc.cc/_updown)   |

- 因服务器规划调整，旧北京入口对应的机器（bj.permod.hitmc.cc 和 bj2.permod.hitmc.cc）已停止使用，为了减少客户端的改动，旧入口域名保留，实际指向新的入口。
- `bj.permod.hitmc.cc` -> `n.permod.hitmc.cc` （建议偏北（<u>**N**</u>orth）方用户使用）
- `bj2.permod.hitmc.cc` -> `s.permod.hitmc.cc` （建议偏南（<u>**S**</u>outh）方用户使用）
