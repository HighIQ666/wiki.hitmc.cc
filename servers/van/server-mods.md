# 服务器端 mod

尽管服务器是原版生存，但是服务器也加入了一些优化等相关的 mod。

截止到当前时刻（2026-01-19），目前已安装的 MOD 如下

- Appleskin  苹果皮，给客户端安装了的玩家提供支持
- Bluemap  大地图
- Bluemap marker manager  大地图标记
- Carpet  请输入文本
- Carpet tis addition 一个 carpet 附属，提供了一些功能，具体可查阅 https://carpet.tis.world/zh-Hans/docs
- Essential addons  一些给 carpet 的便利命令，记得配置 config
- Guglecarpetaddition 假人相关扩展
- Servux  服务端MiniHUD支持，精准放置协议支持等

## 框架/前置/依赖类 MOD

- Silk  前置
- Fabric api  前置，请输入文本
- fabric-language-kotlin 前置
- collective  连锁砍树前置

## 性能优化类 MOD

- Ferritecore  铁氧体磁芯，优化类 mod
- Lithium  锂，优化类 mod
- C2me  区块加载优化

## 体验优化类 MOD

- Jade  请输入文本，为客户端安装的玩家提供支持
- Quickshulker  快捷潜影盒等
- tree harvester  下蹲连锁砍树

## 管理类 MOD

- Ledger  查熊
- Luckperms  权限配置
- Spark  服务器性能检测 mod
